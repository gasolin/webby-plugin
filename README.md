Webby-plugin
============
Hubot plugin template written with plain javascript, compatible with webbybot.

## Installation

* Run the `npm install` command.

```
$ npm install webby-plugin
```

* Add the following code in your `external-scripts.json` file.

```
["webby-plugin"]
```

## Sample Interaction

```sh
Hubot> Hubot ping
Hubot> pong
```
